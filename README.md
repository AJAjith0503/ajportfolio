## ⚙️ Local Development

To run this portfolio application locally on your computer, follow these simple steps:

1. **Clone this repository:**
   ```bash
   git clone https://github.com/AJAjith0503/ajportfolio.git
   cd ajportfolio
   ```

2. **Install all required dependencies:**
   ```bash
   npm install
   ```

3. **Start the Next.js development server:**
   ```bash
   npm run dev
   ```

4. **Compile a production build:**
   ```bash
   npm run build
   ```
